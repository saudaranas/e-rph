🔗 Sambung Google Apps Script

Masukkan URL Web App GAS anda untuk simpan data RPH ke Google Sheets secara automatik.

Cara deploy GAS:
1. Buka Code.gs yang disediakan dalam Google Apps Script
2. Klik Deploy → New Deployment → Web App
3. Tetapkan Execute as: Me dan Who has access: Anyone
4. Copy URL dan tampal di sini
