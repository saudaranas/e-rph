🔗 Sambung Google Apps Script

Masukkan URL Web App GAS untuk simpan data ke Google Sheets secara automatik.

Cara deploy GAS:
1. Buka Code.gs dalam Google Apps Script
2. Klik Deploy → New Deployment → Web App
3. Execute as: Me  |  Who has access: Anyone
4. Copy URL → tampal di sini → klik Sambung
